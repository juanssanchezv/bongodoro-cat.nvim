# Changelog

## v0.2.0

- Added Pomodoro timer overlay with `start`, `pause`, `pause_resume`, `resume`, `stop`, and `status` commands.
- Added optional Pomodoro keymaps, disabled by default.
- Added Neovim health check via `:checkhealth bongo_cat`.
- Added GitHub Actions CI for smoke test and health check.
- Added README demo SVG and GitHub issue templates.
- Disabled diagnostic error animation by default to avoid noisy typing feedback.

## v0.1.0

- Initial public release.
- Added Bongo Cat typing animation with left/right frames.
- Added randomized idle animation and sleep animation.
- Added save reaction and manual/optional error reaction.
- Added floating window renderer, plugin commands, README, MIT license, and smoke test.
